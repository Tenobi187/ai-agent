# Поисковой агент по вашим документам. 

Выберите файл, загрузите его и задайте свой вопрос. 
После того, как выбрали файл, обязательно нажмите кнопку "загрузить" и дождитесь уведомления в чате о загрузке файла. Вы можете загрузить несколько файлов одновременно. 
Чтобы избежать конфликтов в информации, удаляйте файлы.

## Установка
1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Запустите агента: 
```bash
python main.py
```
4. Откройте страницу в браузере
http://localhost:8000/
